# AnyPing

A tiny network diagnostic tool for Windows.

![screenshot](screenshot-v1.4.1.0.png)


## System Requirements

* .NET Framework 4.8


## Features

* IPv4 and IPv6
* DNS lookup
* ICMP Ping
* TCP Ping
* HTTP Ping (using system proxy settings)
* HTTPS Ping (using system proxy settings)
* ICMP Traceroute
